This is the index page for NEPSE-api

Use the following endpoints to access the data

API endpoints

prepend 'https://the-value-crew.github.io/nepse-api/' to all endpoints
  • - `/data/info.json` : Brief info on APIs
  • - `/data/companies.json`: Listed companies on Nepse
  • - `/data/date/{{YYYY-MM-DD}}.json`: Daily stocks data by date
  • - `/data/date/latest.json`: Latest data by date
  • - `/data/company/{{company-code}}.json`: Daily data by company code

How does it work

Nepse operates Sunday to Thursday, from 11:00 AM - 3:00 PM. Each day last 7 days' data is scraped at 10:00 AM, 3:05 PM, and 12:00(midnight). Since Nepse's server crashes frequently, scraping last 7 days' data till today, multiple times a day, seems good approach.
